# 第九章 指针与引用

指针是 C++ 从 C 语言继承的核心机制，也是许多初学者的第一个难关。理解指针意味着理解程序如何与内存交互——变量存储在哪里、如何通过地址间接访问数据、如何在运行时动态申请和释放内存。

引用是 C++ 在指针基础上提供的一种更安全、更简洁的间接访问方式。在现代 C++ 中，引用在大多数场景下可以替代指针，但理解指针仍然是掌握 C++ 的必要条件。

本章涵盖以下内容：

- **指针基础** — 地址、解引用、指针运算、空指针
- **引用** — 左值引用、引用与指针的对比、引用的应用场景
- **动态内存分配** — new/delete、堆与栈、内存泄漏
- **const 与指针** — 指向常量的指针、常量指针、顶层/底层 const
