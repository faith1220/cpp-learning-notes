# 第十九章 底层知识补充

本章作为 C++ 学习的拓展与补充，专门用于收录那些在主线学习中可能会遇到、但为了保持阅读流畅性而不宜在主线中过度展开的计算机底层基础知识。

> **何时阅读本章？**
>
> 即使你跳过本章，也完全不影响你继续学习 C++ 的语法和高级特性。但当你对“为什么代码要这样写”、“数据在内存中到底是什么样子”感到好奇时，或者当你开始接触网络编程、硬件交互等底层领域时，本章的知识将是你不可或缺的基石。
