# 第八章 运算符重载

C++ 允许为自定义类型重新定义运算符的行为，使得自定义类型可以像内置类型一样使用 `+`、`-`、`==`、`<<` 等运算符。这种机制称为**运算符重载**（Operator Overloading）。

合理的运算符重载能让代码更直观——例如两个向量相加写成 `v1 + v2` 比 `v1.add(v2)` 更自然。但滥用运算符重载会让代码难以理解，因此需要遵循一定的规则和原则。

本章涵盖以下内容：

- **重载的基本规则** — 语法、限制、成员函数与非成员函数的选择
- **常见运算符重载** — 算术、比较、赋值、下标、递增递减、流插入/提取
- **类型转换运算符** — 自定义类型与其他类型之间的隐式和显式转换
