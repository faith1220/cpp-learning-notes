# 第六章 类与对象

前几章介绍的变量、函数、数组等工具足以编写功能性的程序，但随着程序规模增长，单纯用函数组织代码会变得力不从心。**面向对象编程**（Object-Oriented Programming, OOP）提供了一种更贴近人类思维的代码组织方式：将数据和操作数据的行为封装在一起，形成**类**（Class）；类的具体实例称为**对象**（Object）。

本章涵盖以下内容：

- **类的定义** — 如何定义类、声明成员变量和成员函数
- **构造函数与析构函数** — 对象的创建与销毁过程
- **访问控制** — public、private、protected 的含义与用法
- **this 指针** — 对象如何引用自身
- **静态成员** — 属于类而非对象的数据和方法
