# 第一章 认识 C++

C++ 是一门历史悠久且应用广泛的编程语言。从操作系统内核到游戏引擎，从数据库系统到高频交易平台，C++ 的身影无处不在。学习 C++ 不仅能让你掌握一门强大的编程工具，还能帮助你深入理解计算机的底层运作方式。

本章将从以下几个方面帮助你迈出第一步：

- **C++ 的历史与发展** — 了解这门语言从何而来，经历了怎样的演变
- **开发环境搭建** — 在你的计算机上安装编译器和编辑器，准备好编程所需的工具
- **第一个程序** — 编写经典的 Hello World 程序，体验完整的开发流程
- **编译与运行** — 理解从源代码到可执行程序的转换过程
