# 第七章 继承与多态

第六章介绍了类与对象的基本机制——封装。本章将讨论面向对象编程的另外两大支柱：**继承**（Inheritance）和**多态**（Polymorphism）。

继承允许在已有类的基础上构建新类，复用已有代码的同时扩展新功能；多态则让程序在运行时根据对象的实际类型选择正确的行为，是构建灵活、可扩展架构的核心手段。

本章涵盖以下内容：

- **继承** — 基类与派生类、继承方式、构造与析构顺序
- **虚函数与多态** — 动态绑定、虚函数表、override 与 final
- **抽象类与纯虚函数** — 接口设计、不可实例化的基类
- **多重继承** — 多基类继承、菱形继承问题与虚继承
